# Kotlin Basics Learning Project

A comprehensive collection of Kotlin programs demonstrating fundamental concepts, data structures, and programming techniques.

## Features

- Array operations and manipulation
- Object-oriented programming with constructors
- Control flow structures
- User input handling
- Matrix operations
- Variable declarations and types
- Mathematical operations (factorial)
- Array algorithms (finding maximum)
- Type conversion techniques
- User-defined functions
- Conditional expressions (when)
- Even and odd number checking

## Project Structure

```
src
├── Array.kt                    # Array operations and examples
├── Class_Constructor.kt        # Class creation and constructors
├── EvenOdd.kt                 # Even/odd number determination
├── Input.kt                   # User input handling
├── Matrix.kt                  # Matrix operations
├── Variables.kt               # Variable types and declarations
├── factorial.kt               # Factorial calculation
├── maximumFromArray.kt        # Finding maximum in arrays
├── typeConversion.kt          # Type casting and conversion
├── userDefinedFunction.kt     # Custom function creation
└── whenExpression.kt          # When expression usage
```

## Getting Started

### Prerequisites
- IntelliJ IDEA or Android Studio
- Kotlin compiler
- JDK 8 or higher

### Running the Programs

1. Clone the repository
2. Open project in IntelliJ IDEA or Android Studio
3. Navigate to any `.kt` file
4. Right-click and select "Run"
5. Or use the green play button in the IDE

### Running from Command Line

```bash
# Compile a Kotlin file
kotlinc filename.kt -include-runtime -d filename.jar

# Run the compiled program
java -jar filename.jar
```

## Learning Path

1. Start with **Variables.kt** - Understand basic declarations
2. Move to **Input.kt** - Learn user input handling
3. Practice **EvenOdd.kt** - Master conditional logic
4. Explore **Array.kt** - Work with collections
5. Study **userDefinedFunction.kt** - Create reusable code
6. Learn **Class_Constructor.kt** - Object-oriented basics
7. Try **whenExpression.kt** - Advanced control flow
8. Practice **typeConversion.kt** - Type handling
9. Implement **factorial.kt** - Recursion concepts
10. Work on **Matrix.kt** - 2D arrays
11. Solve **maximumFromArray.kt** - Algorithm practice

## Key Takeaways

- Kotlin is concise and expressive
- Type safety prevents runtime errors
- Null safety is built into the language
- Functions are first-class citizens
- Extension functions add functionality to existing classes
- When expressions are more powerful than switch statements
- Data classes reduce boilerplate code
